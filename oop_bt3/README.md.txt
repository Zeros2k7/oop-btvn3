# Bài Tập Nhận Diện Lớp Từ Thực Tế (Python)

Dự án này mô phỏng việc chuyển đổi các thực thể thực tế thành lập trình hướng đối tượng (OOP) bằng ngôn ngữ Python.

## Mô tả bài tập
Dựa trên hình ảnh bài tập, dự án xây dựng 3 lớp cơ bản:
1. **Con chó (Dog)**: Quản lý thông tin tên, giống và các hành động như sủa, chạy.
2. **Ô tô (Car)**: Quản lý thông số kỹ thuật và các chức năng vận hành.
3. **Tài khoản (Account)**: Mô phỏng các giao dịch ngân hàng cơ bản.

## Yêu cầu hệ thống
- Python 3.x

## Cách chạy code
```bash
python main.py
